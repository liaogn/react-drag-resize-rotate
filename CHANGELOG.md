# Changelog

## 1.0.0

### Major Changes

- Release the React version of `@liaogn/react-drag-resize-rotate`.
- Reuse the framework-independent core geometry, drag, resize, rotate, flip, cursor, and boundary logic from the Vue 3 package.
- Keep the Vue package API shape for migration parity, including `childrens`, `resizeable`, `rotateable`, `fliped`, `limitX`, and `limitY`.
- Add Vite library builds, TypeScript declarations, CSS export, Docusaurus docs, and a Vite React playground.
