# Props

React keeps the same API names as the Vue package, including `childrens`, `resizeable`, and `rotateable`.

Main props: `w`, `h`, `x`, `y`, `r`, `z`, `bg`, `lock`, `active`, `activeable`, `draggable`, `resizeable`, `rotateable`, `sticks`, `childrens`, `childWrapAttr`, `overflow`, `stickHoverRender`, `limitX`, and `limitY`.
